# ATS CV Builder

A clean, cross-platform Flutter application for generating ATS-friendly (Applicant Tracking System) Resumes/CVs.

## Features
- Comprehensive forms for personal information, work experience, education, and skills
- Dynamic lists allowing multiple entries for experience, education, and skills
- Clean, single-column ATS-friendly CV output
- Web, Desktop, and Mobile responsive

## Setup & Run
1. Ensure you have Flutter installed (version 3.7.2 or higher).
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to start the app.
